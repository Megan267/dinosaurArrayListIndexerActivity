# 🦖 Dinosaur Tracker
 
> *A cozy little C# console app for researchers living in a world where dinosaurs never went extinct!* 🌿🥚
 
Every day brings a new discovery: a stomping herbivore, a sneaky little omnivore, or maybe a terrifying carnivore hiding in the ferns. This app helps you log, browse, and manage all your dino finds in one friendly system. 🔎
 
---
 
## ✨ What This App Does
 
Keep track of every dinosaur you discover with the following details:
 
| Field | Description |
|-------|-------------|
| 🆔 `DinoID` | Unique ID number for each dino |
| 🍖 `DietType` | Carnivore, Herbivore, or Omnivore |
| 🏷️ `Nickname` | The fun, common name (e.g. "Rex") |
| 🧬 `Species` | The scientific name (e.g. *Tyrannosaurus rex*) |
| ⚖️ `Weight` | Weight in kilograms |
 
---
 
## 🛠️ Built With
 
- **C#** 💙
- **ArrayLists**, to store all discovered dinosaurs in one flexible collection 📦
- **Indexers**, for quick, easy access to dino records, just like using `dinoTracker[i]` 🗂️
